#### 🚀 Быстрый старт

Всего 4 команды — и приложение запустится на твоём телефоне или в браузере.

### 📋 Требования

- [Node.js](https://nodejs.org/) (версия 18 или выше)
- [Git](https://git-scm.com/)
- [Expo Go](https://expo.dev/go) на телефоне (iOS/Android) — опционально

### 🔧 Установка и запуск

```bash
# 1. Клонируй репозиторий
git clone https://github.com/ТВОЙ_USERNAME/FlashMind.git

# 2. Перейди в папку проекта
cd FlashMind

# 3. Установи зависимости
npm install
# или если ты пользуешься yarn:
# yarn install

# 4. Запусти Expo
npx expo start
</div>
